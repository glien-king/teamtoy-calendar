
# Calendar 使用说明
![enter image description here][1]

##### 使用方法:

1.下载代码到plugin目录下，然后到插件菜单中，开启即可

2.添加TODO后，点击某一个TODO，设置时间和优先级

3.到日历菜单中查看

##### 注意事项:
1.对于旧的TODO(插件安装之前就已经存在的TODO),也会在日历中显示,以创建时间和最后操作时间为开始和结束时间.

2.建议安装插件后,所有新的TODO都设置时间和优先级.
